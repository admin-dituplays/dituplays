Опис майданчика

Будиночок Спортивний — це ідеальний вибір для активного розвитку вашої дитини. Модель поєднує спортивний комплекс та затишний ігровий простір.

Комплектація включає:

  • Закриту пісочницю;
  • Меблі в будинку для затишного відпочинку;
  • Еластичну гойдалку та гойдалку "гніздо лелеки";
  • Руколазку, турніки, гімнастичні кільця та бруси для тренувань;
  • Скеледром, пожежну трубу, гірку для активних ігор.

Усі елементи виготовлені з екологічних матеріалів: масиву сосни камерної сушки, букових нагелів, фарб на водній основі Kemichal та бельгійських комплектуючих. Конструкція продумана для забезпечення повної безпеки дитини під час гри та тренувань.
